# Meme generator

Generate memes

# Get started

## Software dependencies	

- Python 3.7

- xpdf

  - On linus:

    ```bash
    sudo apt-get install -y xpdf
    ```

  - On mac:

    ```bash
    brew install xpdf
    ```

    

    

## Install Python dependencies

```bash
pip install -r requirements.txt
```
